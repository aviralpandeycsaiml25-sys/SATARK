🚨 SATARK – Crime Awareness & Emergency Response Portal
🧭 Overview

SATARK is a data-driven crime awareness and emergency response web portal designed to improve public safety by providing crime insights and real-time SOS support.

The platform helps users:

View historical crime trends

Identify high-risk areas

Trigger SOS alerts in emergencies

Access awareness-based safety insights

🎯 Problem Statement

Public access to structured crime data is often limited and not user-friendly.
There is also a lack of integrated emergency alert systems within awareness platforms.

SATARK aims to bridge this gap by combining:

Crime data visualization

Structured reporting

Emergency SOS functionality

💡 Solution

SATARK provides:

📊 Crime data dashboard

📍 Area-based crime insights

🚨 Emergency SOS alert feature

📈 Historical crime trend display

The system organizes publicly available crime datasets into an interactive and accessible format.
